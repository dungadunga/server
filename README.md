# dungadunga-server

## 구동 시 필요사항

- 명령어

  > npm install  
  > npm run start

- .env

  NODE_SETTING

  > NODE_ENV= production | development  
  > PORT= number  
  
  DATABASE_SETTING

  > DB_HOST= string  
  > DB_NAME= string  
  > DB_USERNAME= string  
  > DB_PASSWORD= string

  AWS_SETTING

  > AWS_ACCESSKEY_ID= string 
  > AWS_SECRET_ACCESSKEY= string 
  > AWS_REGION= ap-northeast-2

  FACEBOOK
  
  > FACEBOOK_CLIENT_ID= string
  > FACEBOOK_CLIENT_SECRET= string 

  GOOGLE

  > GOOGLE_AZP= string
